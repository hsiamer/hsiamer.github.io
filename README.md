noveldef.py的用法

打开网站 https://www.-------.com


  定制工具,不通用,真实网址自行寻找


找到自己想要下载的小说并打开页面

当前小说的地址为

https//www.-------.com/book/$bookid/xxxxxx

记住其中的 $bookid 内容

python3执行 noveldef.py

python3 noveldef.py

按照提示输入上一步获取的 bookid

回车后,回显书籍名称,总章节数

输入起始章节,如果要从第一章开始,输入1

回车继续

自动提交远程仓库的功能,需要设置 Clone with SSH 

具体方法自行百度
